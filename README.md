# ENG-DRB
ENG‑DRB is the first PDTB‑style discourse relation corpus drawn from hands‑on engineering tutorial video transcripts (civil, mechanical, and electrical/electronics). It contains dense causal and temporal relations, a high proportion of explicit senses, and frequent overlapping/embedded structures.
Link to paper: https://aclanthology.org/2025.findings-ijcnlp.81/. The dataset can be accessed at: https: //doi.org/10.57967/hf/6895.

This reposite shows all the benchmarking codes used in the experiment.
